# Online Quiz System

## Overview

The Online Quiz System is a comprehensive platform designed to facilitate the creation, management, and participation in quizzes. The system offers various features tailored for both users and administrators, including authentication, quiz management, real-time quiz participation, and detailed results analysis.

## Features

### Authentication Module
- **Login**: Secure user login functionality.
- **Registration**: New user registration process.
- **Forgot Your Password**: Option to reset the password if forgotten.
- **Reset Your Password**: Allows users to reset their password.
- **User Profile**: Manage account details and edit profile information.
- **Admin Profile**: Includes additional functionalities for admin users such as viewing and managing users, and adding new admin users.

### Quiz Completion Module
- **Quizzes**: Support for both offline and live quizzes.
- **Timer**: Each quiz has a timer, automatically submitting the quiz once the time runs out.
- **Types of Questions**:
  - Single choice question
  - Multiple choice question
  - Review needed question
  - Written answer question
  - True/False question
- **Navigation Buttons**:
  - First question
  - Previous question
  - Next question
  - Last question
  - Submit answers (only on the last question)

### CRUD Quiz Module
- **Create, Read, Update, Delete (CRUD)**: Full support for managing quizzes, including creation, modification, and deletion.

### Live Quiz Module
- **Live Quiz Creation**: Facilitates real-time quiz creation and management.
- **User Participation**: Users can join live quizzes and wait for the creator to start the quiz.

### Results and Statistics Module
- **My Taken Quizzes**: View history and details of quizzes taken, including answers, grading, and correct answers.
- **Results Visualization**: Colors used for answer validation:
  - Green: Correct
  - Light green: Correct but not picked
  - Red: Wrong
  - Yellow: Partially correct
  - Teal: Review-needed
- **Email Results**: Option to receive quiz results via email.
- **Created Quiz Stats**: Quiz creators can see results for all users who took the quiz, and can obtain results from a specific period via email.

### User Interface Pages
- **Home Page**: Landing page with general information and navigation.
- **Quizzes Page**: List of available quizzes with the option to start a game.
- **Quiz Details Page**: Detailed information about individual quizzes.
- **Profile Page**: User profile with access to personal information and quiz creation.
- **Create Quiz Page**: Interface for creating new quizzes, either manually or using AI-generated questions.
- **My Quizzes Page**: List of quizzes created by the user, with options to modify or delete them.

## Getting Started

To get started with the Online Quiz System, follow these steps:
1. **Register an Account**: Sign up using the registration page.
2. **Login**: Access your account by logging in.
3. **Create a Quiz**: Navigate to your profile and select "Create Quiz" to start building your quiz.
4. **Participate in Quizzes**: Choose from available quizzes and start playing.
5. **View Results**: After completing quizzes, view your results and statistics in the "My Taken Quizzes" section.
